    계정 생성

    계정을 생성하여 다양한 인공지능을 이용할 수 있습니다.

    1. 구글 계정으로 로그인
    2. 회원가입 바로가기

      홈페이지(refactoring.clickai.ai)에 접속한 후, 구글 계정으로 로그인 또는 회원가입 바로가기 를 클릭하세요.

    1. 회원 가입

    • 원하시는 회원 가입 방법을 클릭하여 회원 가입이 가능합니다.

    1-1. 구글 계정으로 로그인

    1-2. 회원가입 바로가기

    1) 필요 정보 입력 하기

    정보: 계정 생성에 필요한 입력 정보를 입력해 주세요.

    1. 이메일
      • 사용 가능한 학교 및 회사 이메일을 입력합니다. 개인 이메일은 사용이 불가합니다.
      • 회원 가입을 위해서는 중복확인을 해야합니다.

    2. 비밀번호
      • 비밀번호는 영문, 숫자, 특수문자 3종류를 조합하여 최소 8자리 이상의 길이로 구성하여야 합니다.

    3. 이용약관, 개인정보보호 약관
      • 계정 활성화를 위하여 체크해주시면 됩니다.

    * 이메일 인증하기
